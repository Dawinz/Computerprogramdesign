## 🎯 What is Factos?

Factos is an **AI-driven platform** that automates the entire SaaS product development lifecycle through an intelligent **8-step pipeline**:

1. 💡 **Planning** - AI-powered requirement gathering
2. 🔍 **Extraction** - Structured data extraction
3. 🎨 **Design** - Design system creation
4. 🏛️ **Architecture** - System architecture design
5. 💻 **Development** - Live code generation
6. 🧪 **Testing** - Multi-agent concurrent testing
7. 🔒 **Security** - Security validation
8. 🚀 **Deployment** - Automated deployment

---

## 🚀 Quick Start (3 Steps)

### 1. Install Dependencies
```bash
pnpm install
# or
npm install
```

### 2. Start Development Server
```bash
pnpm dev
# or
npm run dev
```

### 3. Open in Browser
```
http://localhost:5173
```

**That's it!** You're now running Factos locally. 🎉

## 🎯 The 8-Step Pipeline

Each step has its own view component:

1. **PlanningView** - Chat-based AI planning interface
2. **ExtractionView** - Data schema extraction and preview
3. **DesignView** - Design system builder with live preview
4. **ArchitectureView** - System architecture diagrams
5. **DevelopmentView** - Live code generation interface
6. **TestingView** - Multi-agent testing (10/100/1000 users)
7. **SecurityView** - Security validation dashboard
8. **DeploymentView** - Deployment pipeline and monitoring
